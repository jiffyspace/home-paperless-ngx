# Home paperless-ngx Service

Configuration for [paperless-ngx](https://paperless-ngx.readthedocs.io/en/latest/) service.

# Prerequisites

* Docker
* Traefik

# Setup

* Clone repo
* Create `.env` files - `docker-compose-example.env` -> `docker-compose.env`, `example-env` -> `.env`
* `docker-compose up -d`
